// turbo-all

1. Stage all changed files:
```
git add -A
```

2. Commit with a descriptive message:
```
git commit -m "<message>"
```

3. Push to the remote branch:
```
git push origin tidb-main
```

4. Vercel will auto-deploy from GitHub. Wait ~1-2 minutes for the deploy to complete.
